Project Overview

Michigan Institute for Clinical and Health Research has developed a web protol for users to start or participate medical research. However, the previous website is not responsive nor user friendly for people with disability and thus could not provide good user experience. My team and I are to redesign and improve the website.

Work Flow

Sketch->Interface

Sketch

To start with, my team and I had to do sketches to decide what the page looked like. We decided to use board since it's easy to draw our ideas and convinient to discuess with others and made changes if necessary.


Interface

After sketching, we started to code in html to create basic template that looked like what was on the white board. Then we began adding style sheets and plugins to build basic interactions.(View Beta)
