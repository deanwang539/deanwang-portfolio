Project Overview

Dr.Lars Junghans from Taubman College of Architecture and Urban Planning in University of Michigan received client request to develope a web application which can calculate and evaluate building construction in professional perspectives (sensitivity study, Meteo ratios and etc.). As a lead developer in the team, it is my duty to help achieve the goal.

Work Flow

Interface Prototype->Code Translation->Algorithm Integration->Backend Coding

Interface Prototype

To start with, I talked to Dr.Junghans about his idea of the web applications and expected features. Later I came up with the interface prototype and integrated it with HTML/CSS/Javascript.


C++ Code Translation

Dr.Junghans had many unique algorithms in C++ to help calculate building parameters such as internal and external temperature. Before integrating them to our web applications, my team and I have to translate them into Java.


Backend Coding

I chose to use MyEclipse for better support of Web Service Project. Except for algorithm integrations, I am also in charge of Jave Servlet coding to ensure correct request/response contents displaying on the Web Application. In addition, I am responsible for coding database connections and inqueries in Java as well.
